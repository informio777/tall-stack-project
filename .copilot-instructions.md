# GitHub Copilot Instructions for TALL Stack Project

## Project Context
This is a TALL Stack project using:
- **T**ailwind CSS for styling
- - **A**lpine.js for JavaScript interactions
  - - **L**aravel as PHP framework
    - - **L**ivewire for dynamic components
      - - **TallStackUI** for UI components
        - - **Vite** for asset bundling
          - - **Pusher** for real-time features
           
            - ## Code Style Guidelines
           
            - ### PHP/Laravel
            - - Use strict typing: `declare(strict_types=1);`
              - - Follow PSR-12 coding standards
                - - Use Laravel conventions for naming
                  - - Add detailed docblocks for all methods
                    - - Use type hints for all parameters and return types
                     
                      - ### Livewire Components
                      - - Use kebab-case for component names
                        - - Add `#[Layout]` attribute for custom layouts
                          - - Use `#[Validate]` for form validation
                            - - Implement proper lifecycle hooks
                             
                              - ### JavaScript/Alpine.js
                              - - Use Alpine.js directives: `x-data`, `x-show`, `x-if`
                                - - Keep JavaScript minimal and declarative
                                  - - Use Alpine stores for shared state
                                   
                                    - ### CSS/Tailwind
                                    - - Use Tailwind utility classes
                                      - - Follow mobile-first responsive design
                                        - - Use TallStackUI components when available
                                          - - Create custom components in `resources/css/components/`
                                           
                                            - ## File Structure Patterns
                                            - - Controllers: `app/Http/Controllers/`
                                              - - Livewire: `app/Livewire/`
                                                - - Models: `app/Models/`
                                                  - - Views: `resources/views/`
                                                    - - Components: `resources/views/components/`
                                                     
                                                      - ## Common Patterns
                                                      - - Use Livewire for dynamic forms
                                                        - - Implement real-time updates with Pusher
                                                          - - Use TallStackUI components for consistency
                                                            - - Follow Laravel best practices for security
